### Design Rules (STRICTLY ENFORCE)

1. **NO glassmorphism** — this is heavy industry, not SaaS
2. **NO bento grids** as default layout
3. **NO generic corporate stock photos** — generate industrial imagery or use abstract
4. **NO rounded corners everywhere** — keep edges sharp (2-4px max) for precision feel
5. **NO pastel colors** — bold red, hard charcoal, clean white only
6. **YES diagonal section dividers** — angled transitions between sections (Rockwell-style)
7. **YES red accent bars** — horizontal accent stripes to break sections
8. **YES bold typography** — Inter 800 Black for headings, strong visual hierarchy
9. **YES scroll-triggered animations** — staggered reveals for stats, product cards, client logos
10. **YES animated counters** — stats count up when scrolled into view (30+, 40+, 10.4m)
11. **YES full-bleed hero images** — industrial photography with dark overlays
12. **YES "Request a Quote" CTAs** — red buttons on every product section

## ⚠️ COMMON MISTAKES TO AVOID

1. **Don't use placeholder text.** Write REAL, compelling copy for every section.
2. **Don't make it look like a template.** Every section should feel custom and premium.
3. **Don't forget mobile.** Design mobile-first. Test at 375px width.
4. **Don't skip animations.** The site must feel alive and modern. Static = failure.
5. **Don't use generic stock photos.** Generate custom industrial visuals.
6. **Don't hardcode the year.** Use `new Date().getFullYear()` in the footer.
7. **Don't ignore spec table mobile UX.** Tables must scroll horizontally on mobile.
8. **Don't forget accessibility.** Keyboard nav, reduced motion, contrast ratios.
9. **Don't make the RFQ form too long.** Only essential fields required, rest optional.
10. **Don't mix color temperatures.** Stick to red + charcoal + white. No blue, no green.
